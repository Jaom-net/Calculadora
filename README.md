
# Calculadora WPF

Este proyecto es una práctica de desarrollo de aplicaciones de escritorio utilizando **.NET** y **WPF**. 
El objetivo principal ha sido replicar, de una manera muy básica la funcionalidad y el diseño de la calculadora estándar 
de Windows, implementado la lógica en C# y el diseño en XAML.

## Funcionalidades

Actualmente la calculadora soporta:
* **Operaciones básicas:** Suma, resta, multiplicación y división.
* **Manejo de decimales:** Uso correcto del punto flotante.
* **Validaciones:** Control de errores básicos (como la división por cero).
* **Interfaz:** Diseño limpio utilizando Grid y Estilos en XAML.

## Tecnologías

* C#
* WPF (.NET 10.0)
* Visual Studio 2026

## Instalación y Uso

Simplemente clona el repositorio y abre la solución en Visual Studio:

1.  `git clone https://github.com/Jaom-net/Calculadora.git`
2.  Abre el archivo `.sln`
3.  Ejecuta el proyecto con `F5`.
